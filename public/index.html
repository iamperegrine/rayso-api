Create beautiful code screenshots using ray.so via REST API.

API Endpoint

POST/GET /api

Usage

Send a GET or POST request to /api with your code and optional parameters.

Example POST Request:

{
  "code": "console.log('Hello World');",
  "language": "javascript",
  "title": "My Code",
  "theme": "candy"
}

Example GET Request:

/api?code=console.log('Hello')&language=javascript&title=Test

Parameters

  • code (required): The code to screenshot
  • title: Title for the code box (default: "Untitled-1")
  • theme: Color theme (default: "breeze")
  • language: Programming language (default: "auto")
  • background: Include background (default: true)
  • darkMode: Use dark mode (default: true)
  • padding: Padding size - 16, 32, 64, or 128 (default: 32)
